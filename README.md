# @bonuz/seo

A simple **zero dependency** SEO package.

## ⭐️ Features
---
- 🤖 Robots.txt
  - ✅ Create configuration with built in validation.
  - 📦 Create preconfigured settings for allow and disallow all.
- 🏷️ Robots Meta Tag
  - 📦 Create preconfigured settings for allow and disallow all.
